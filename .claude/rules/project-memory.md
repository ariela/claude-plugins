### ディレクトリ構造の意図

```
.claude/
├── commands/    # スラッシュコマンド定義
├── hooks/       # セッションフック
└── rules/       # プロジェクトルールとメモリ（このディレクトリ）
```

**設計意図:**
- ClaudeCodeの設定を一元管理
- ドキュメントベースの知識管理
- 再利用可能なコマンドとフックの提供
